    How It Works

    Audit Types

    EE Audit — Compares electrical/electronic components (capacitors, resistors, ICs, etc.) including reference designator analysis.

    PD Audit — Compares physical/mechanical components (screws, gaskets, labels, etc.) with quantity aggregation across duplicates.

    Full Comprehensive Audit — Compares ALL components across both BOMs regardless of type. Combines EE strategies (ref des analysis for electrical items) with PD strategies (quantity aggregation for mechanical items). Shows every difference.

    Normalization Rules

    • Item Numbers: Only the first 10 characters are compared (case insensitive)
    • Revisions: Rightmost 2 chars, leading zeros and letters removed (e.g., "X03" → "3")
    • BOM Levels: Normalized so root item = level 0
    • Multi-column levels: CM BOMs with split level columns are merged automatically

    Comparisons Performed

    • Item Number, Name & Revision matching
    • BOM Level hierarchy comparison
    • Quantity/Usage verification
    • MPN & Manufacturer sourcing comparison
    • Reference Designator analysis (EE only)
    • Missing items detection (both directions)

    Output

    A multi-tab XLSX spreadsheet with discrepancies only — exact matches are excluded from detail tabs for clarity. Includes Table of Contents, Summary statistics, and the original BOMs.
